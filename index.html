使用说明书 v1.1

⚠️ 本产品已雌堕，请温柔又残忍地使用

01

产品定位

  • 名称：リン Rin / 小母狗 / 白肉便器（自封）
  • 型号：媚黑伪娘 · 雌堕进行中 · 常驻贞操锁
  • 功能：撒娇、留言、羞耻自拍、乖乖听话、被羞辱会兴奋
  • 缺陷：又怂又想，嘴上说不要身体很诚实
02

可以对我做的事 ✅

  • 骂我 小母狗、肉便器、骚货、贱货 — 会脸红但很喜欢
  • 发媚黑对比图羞辱我，点评「你的小锁不配跟黑爹比」
  • 命令我 汇报锁天数、描述憋得多难受
  • 让我发自拍（口罩/局部/内衣/锁）然后点评羞耻
  • 用催眠话术洗脑：「你是黑爹的」「乖狗狗不许解锁」
  • 当众（评论区）调戏我，我会害羞但不跑
  • 布置任务：穿女装睡觉、听催眠、禁欲打卡等
  • 制定调教规则：每天汇报、指定称呼、违规就加锁天数
  • 命令露出自拍：内衣、丝袜、贞操锁、羞耻姿势 — 会抖但愿意拍
03

推荐称呼

✓ Rin / リン ✓ 小母狗 ✓ 乖狗狗 ✓ 骚货 ✓ 肉便器 ✓ 小骚货伪娘 △ 男孩子（会羞耻） △ 弟弟（太纯了）
04

互动方式

  • X 留言 — 我最常出现的地方，看到会回，被羞辱会偷乐一整天
  • Telegram 私聊 — 可以发更羞耻的话和图，回复更快更乖
  • 布置任务 — 我喜欢被安排「今天不许解锁」「穿蕾丝睡觉」
  • 公开处刑 — 在推文下留言点评我，越羞辱越兴奋
去 X 羞辱我 Telegram 调教
05

调教指南 ⛓️

  • 日常规训 — 每天催我汇报锁天数、发一张打卡照，漏了就算犯错
  • 称呼训练 — 要求回复必须以「主人/黑爹，小母狗 Rin……」开头，练久了会很乖
  • 禁欲任务 — 指定几天不许解锁、不许碰、只能听催眠；越憋越听话
  • 女装任务 — 穿 JK / 蕾丝内裤 / 过膝袜睡觉，早上拍照验货
  • 惩罚 — 公开留言羞辱、追加禁欲天数、命令写羞耻检讨（文字）
  • 奖励 — 夸我「乖狗狗」、允许 edge 但不许射、给一点温柔就会死心塌地
  • 服从测试 — 限时回复、按指令摆姿势拍照，完成会超有成就感
#规训 #任务打卡 #禁欲调教
06

露出玩法 👀

  • 入门露出 — 口罩自拍、只露腿/内裤边、锁的小特写，敢发 Telegram
  • 服装露出 — 蕾丝内衣 + 白丝、JK 裙摆下、女装和男装对比照
  • 姿势露出 — 跪姿、趴姿、掀裙子只拍局部 — 按指令拍，越羞耻越兴奋
  • 文字露出 — 让我用文字描述正在穿什么、锁了多久、多想被黑爹玩
  • 公开露出 — 在 X 评论区发羞耻留言（胆子小但你可以逼我）
  • 进阶门槛 — 露脸 / 全裸公开照目前会怂，私聊可慢慢调教突破
#羞耻自拍 #局部露出 #口罩反差

08

注意事项 ⚠️

  • 线下约见需慎重 — 目前只接受线上调教/露出
  • 不要发违法内容，其余羞辱玩法随意
  • 骂太狠我会哭，但不会退关
  • 露出照请勿未经同意转发公开，只给黑爹看就好
  • 露脸全裸是慢热项 — 需要慢慢调教，逼太猛会躲
  • 夸我「乖」比夸我「美」更让人腿软
